# Electric-Vehicle-Data-Analysis
Electric Vehicle Data Analysis is the project which gives insights on the total electric vehicles manufactured, make, model and company analysis. I have designed the interactive dashboard which gives insights on the all the above mentioned features, also this dashboard have different filters to choose to show analysis in more granularity.

This is the link to the dataset used for this project: https://docs.google.com/spreadsheets/d/1AfLoIfsmdgNkoD_C4f9eb2Q5wwufqhjN-kRxKOTydUQ/edit?usp=sharing
